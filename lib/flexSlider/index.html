Responsive background slider thing

FlexSlider will automatically be initialised on elements with the class of .flexSlider but you can set up a custom version as required. To do so dont use the 'flexSlider' classname and call the plugin on dom ready (passing your required settings):

$('#myFlexSlider').flexSlider({
    loop: true, //loop back to start else stop at end
    transitionSpeed : 600, //slide animation speed (milliseconds)
    cycle : 6000, //if >0 automatically cycle through slides every x milliseconds (pauses on hover)
    complete: function(){alert('Slider is ready!')} //optional callback
});

Animated Slide Text
Your text can be animated when the slide comes into view. Style and position the elements as you like then apply one of the following classnames. Don't edit the '.animate-in-from-' styles unless you know what you are doing ;) Also, if you use your own animators you should separarate your style from the animations and give your animation-only styles a classname that begins 'animate-'. This is because the script needs to measure the dimensions of each slide as it will be once tha animation is complete (it therefore removes this classname when getting measuring the slide and re-applies it after.

.animate-in-from-left
.animate-in-from-right
.animate-in-from-top
.animate-in-from-bottom
