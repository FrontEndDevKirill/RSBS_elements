Settings
The map works by structuring your HTML in the following way:

<!-- Main Map Div--> <div id="map1" class="map" data-latlng="-34.397,150.644" data-style='[ { "stylers": [ { "saturation": -100 } ] } ]' data-icon="../../img/marker.png" data-zoom="1"/> <!-- Markers are added like so, within the map div: --> <div class="map-marker" data-latlng="-34.397,150.644" data-icon="../../img/marker2.png" data-title="Marker 1"> Info Window Forever! </div> <!-- Second marker --> <div class="map-marker" data-latlng="-34.600,150.644" data-icon="" data-title="Marker 2"> SECOND INFO WINDOW </div> </div>

The main map div allows the following data attributes:

  • data-latlng - If supplied, this will be the center point of the map (If not supplied, map will default to fit all markers
  • data-zoom - If supplied, this will be the zoom level of the map
  • data-icon - The default icon image
  • data-style - Any styling to apply to the map in the form of a google map stylers JSON object (See useful links below
  • data-disableui - Removes Google's standard interface (zoom, streetview, map/satellite etc.
  • data-scrollwheel- Set to "false" to disable zooming via scrollwheel
  • data-draggable - Set to "false" to disable dragging

Any markers should be included within the main map div containing any information to be included in that marker's info window. They take the following data attributes:

  • data-latlng - Marker location (latlng or address required)
  • data-address- Marker location in address form - will be geocoded by plugin (latlng or address required)
  • data-icon - Overrides the default marker (either Google's or the map's default)
  • data-title - The title of the marker
