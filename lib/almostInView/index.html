Almost In View

By default the plugin applies this method to items with the class .almost-in-view but you can initialise it in the standard jQuery way on any element you like.

$('.me').almostInView({ speed: 1000, //ms (set to zero to use custom css animation)* outOfViewClassName: 'out-of-view', //default classname for if custom css animations are used inViewClassName: 'in-view', //default classname for if custom css animations are used far: 200, //how far the out-of-view item is pushed away (in px) near: 80, //the point at which the item is deemed to be 'almost' in view (in px) });

Basic animation is baked-in using jQuery.animate but you can use custom CSS animations should you wish:
Set speed:0 to disable the built-in animation. The plugin then applies the appropriate classnames as the target elements come into view.
